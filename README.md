## Core skills used in this project

- React.js
- Typescript
- Vite
- React Redux / Redux Toolkit / RTK query
- Tailwind CSS
- React Hook Form / Yup
- ESLint
- Prettier
- Responsive Web Design
- RESTful API
- Git Version Control
